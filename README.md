Содержание

Обзор проекта
NeuroEnglish Tutor — это интеллектуальный Telegram-бот для изучения английского языка, сочетающий структурированные уроки с возможностями искусственного интеллекта.

Ключевые особенности:
Структурированное обучение по категориям (грамматика, фонетика, лексика)
AI-ассистент для объяснения сложных тем (GigaChat/YandexGPT)
Адаптивная система сложности на основе прогресса ученика
Локальная база данных SQLite для хранения прогресса
Геймификация (серии дней, достижения, статистика)

Целевая аудитория:
Студенты, изучающие английский язык

Архитектура системы
main.py - Ядро бота, обработка команд и состояний
database.py - Управление данными пользователей и прогресса
learning_engine.py - Логика обучения и рекомендаций
gigachat_api.py - Интеграция с AI API (или yandexgpt_api.py)
ai_router.py - Роутинг между разными AI-провайдерами

Стек технологий
Python 3.9+ - основной язык разработки
Aiogram 3.10 - асинхронный фреймворк для Telegram ботов
SQLite - встроенная база данных
PyCharm/VSCode - среда разработки
GigaChat API - нейросеть для генерации контента (альтернативы: YandexGPT, OpenAI)

Установка и запуск
Python 3.9 или выше
Аккаунт Telegram и BotFather для создания бота
API ключ GigaChat (или альтернативного сервиса)

Структура базы данных
Основные таблицы:
1. users (Пользователи)
2. exercise_history (История упражнений)
3.  learning_plans (Планы обучения)
   
API интеграции
GigaChat API

Команды бота
/start - Начало работы, регистрация
/menu - Главное меню
/explain [тема] - Объяснить тему через AI
/stats - Статистика прогресса
/reset - Сброс истории AI
